## 📦 Getting Started

### ✅ Prerequisites

- Node.js >= 16.x
- npm or yarn

### 📥 Installation

```bash
git clone https://github.com/YuriiVychivskii/portfolio-next.git
cd portfolio-next
npm install
# or
yarn install
````

### 🚀 Run Locally

```bash
npm run dev
# or
yarn dev
```

Visit [http://localhost:3000](http://localhost:3000)
